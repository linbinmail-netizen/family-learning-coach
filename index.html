# Content Plan This project uses a source-controlled content workflow so learning material is traceable before it appears in the app. ## Official Sources - TEKS machine-readable documentation: https://teks.texasgateway.org/ - STAAR overview: https://tea.texas.gov/student-assessment/staar - STAAR released test questions: https://tea.texas.gov/student-assessment/staar/staar-released-test-questions Important constraints: - STAAR questions are aligned to TEKS. - Recent STAAR released tests are primarily online, not static PDFs. - Released questions, answer keys, rationales, and student expectations should be stored as source references before being turned into app questions. - Do not copy paid commercial platform content into the question bank. ## First Coverage Target Grade 7: - Math: proportional reasoning, rational number operations, expressions, equations, multi-step word problems - RLA: central idea, text evidence, character analysis, theme, vocabulary in context, short constructed response - Science: ecosystems, matter and energy, force and motion, Earth systems, data interpretation - Pre-Algebra: variables, expressions, one-step and two-step equations, slope readiness Grade 8: - Math: proportional and linear relationships, equations, transformations, Pythagorean theorem, data - RLA: compare texts, argument, inference, author purpose, text evidence, written response - Science: matter, energy, force, Earth and space, organisms and environments, experimental design - Algebra I readiness: equations, functions, graph interpretation, word-problem modeling Grade 9: - English I: central idea, text evidence, revision/editing, author's craft, argumentative response - Algebra I: linear functions, equations and inequalities, systems, quadratics, exponential functions - Geometry: congruence, similarity, proofs, circles, coordinate geometry, measurement - Biology: cells, genetics, evolution, ecology, biomolecules, data and scientific reasoning ## Import Stages 1. Build TEKS knowledge map for 7th, 8th, and 9th grade paths. 2. Add 10 diagnostic questions per subject, written in-house and reviewed against TEKS. 3. Add released STAAR source references and item rationales where available. 4. Add AI coach prompts for each knowledge point. 5. Add parent-facing report language for each weakness pattern. ## Quality Rules - Every question must have grade, subject, knowledge point, difficulty, answer, explanation, and coaching hints. - The AI coach may ask questions and give hints, but should not directly reveal the answer first. - Imported or adapted questions must include a source URL and usage note. - Commercial paid content must not be scraped or copied.
